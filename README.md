# PID Simulator

A Python-based simulator for learning and visualizing PID (Proportional–Integral–Derivative) controllers.

## Objectives

* Understand the fundamentals of PID control.
* Implement a reusable PID controller in Python.
* Simulate physical systems such as temperature control.
* Visualize system response using graphs.
* Learn software engineering practices using Git and GitHub.

## Technologies

* Python 3
* NumPy
* Matplotlib
* Git
* GitHub
* Visual Studio Code
